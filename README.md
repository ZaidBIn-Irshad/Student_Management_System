# Student Management System (C++)

## Description

Student Management System is a console-based application developed in C++. The project allows users to manage student records efficiently through a menu-driven interface.

## Features

* Add student records
* Display all student records
* Search a student by Roll Number
* Update student information
* Delete a specific student record
* Delete all records

## Technologies Used

* C++
* Arrays
* Functions
* Loops and Conditional Statements

## How to Run

1. Download or clone the repository.
2. Open the project in any C++ IDE (CodeBlocks, Dev C++, Visual Studio Code).
3. Compile and run `main.cpp`.
